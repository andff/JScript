# 🖥️ SocialGrid v2 — Painel Configurável de Redes Sociais

## 🚀 Como usar

### 1. Instalar Node.js
https://nodejs.org (v18 ou superior)

### 2. Instalar dependências
```bash
npm install
```

### 3. Iniciar
```bash
npm start
```

---

## 🎛️ Tela de configuração

Ao abrir, você verá a tela de configuração onde pode:

**① Escolher os sites**
- Clique nos cards para selecionar/deselecionar
- Sites disponíveis: X, Facebook, Instagram, Reddit, YouTube, TikTok, LinkedIn, Twitch, Pinterest, Discord
- Adicione qualquer site personalizado no campo de texto

**② Escolher o layout**
| Layout | Descrição |
|---|---|
| 1 Coluna | Todos os painéis empilhados verticalmente |
| 2 Colunas | Dois painéis lado a lado |
| 3 Colunas | Três painéis lado a lado |
| 4 Colunas | Quatro painéis lado a lado |
| Grade 2×2 | 4 quadrantes iguais |
| Grade 3×2 | 6 painéis em grade |

---

## 🔄 Voltar para configuração

No painel, clique no botão **⚙ Config** na barra superior para voltar à tela de configuração e escolher outros sites ou layout.

---

## 🔒 Logins salvos

Cada site tem uma sessão separada e persistente — os logins ficam salvos entre sessões.
